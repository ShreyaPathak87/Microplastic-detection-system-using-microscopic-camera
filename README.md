## 💧 Project Overview

The **Microplastic Detection System** is an AI-powered project that analyzes microscopic images of water samples to detect microplastic contamination.

It classifies water samples into contamination levels and determines whether the water is **safe for drinking or not**.

---

## 🎯 Key Features

- 📸 Microscopic image-based detection  
- 🧠 Deep learning model (MobileNetV2)   
- 📊 Confidence score output  
- 🔍 Top prediction probabilities  
- ⚡ Fast Streamlit interface  

---

## 🧪 Output Classes

| Class | Meaning | Water Status |
|------|--------|--------------|
| A | No Microplastic | 🟢 Drinkable |
| B | Low Microplastic | ⚠️ Not Drinkable |
| C | Medium Microplastic | ⚠️ Not Drinkable |
| D | High Microplastic | 🚨 Not Drinkable |
| F | Severe Contamination | ☠️ Not Drinkable |

## ⚙️ How It Works

1. 📤 Upload microscopic water image  
2. 🧠 AI model processes image  
3. 🔍 Predicts contamination level  
4. 📊 Shows confidence score  
5. 💧 Displays drinkable / not drinkable result

## ▶️ Run the Project

### Step 1: Install dependencies
```bash

```

### Step 2: Run the app
```bash
streamlit run main.py
```
